Twitch Plays Pokémon Updater for Firefox

(Chrome version by @Xylem, ported to Firefox by @rctgamer3)

Get live updates from the Reddit Live Updates page as notifications in Firefox.

Steps to install (until i implement a fancy install button here):

  1. Download the latest version
  2. Save it to your computer
  3. Open the Add-ons Manager
  4. Click the cog button and select "Install Add-on From File"
  5. Select the .xpi you downloaded in step 1.
  6. Click the install button in the installation pop-up
  7. Restart Firefox
  8. ????
  9. PROFIT!

Known issue: Due to Firefox limitations (nsIAlertsService) notifications will only be visible for ~4s. This behaviour cannot be changed.
